Ecommerce

Aplicativo Java e Angular em arquitetura de microsserviços. Construindo utilizando os seguintes componentes: Mensageria utilizando Kafka, Observabilidade com Loki, Prometheus e Grafana, Tracing com Micrometer, SQL e NoSQL com PostgreSQL e MongoDB, Containerização com Docker. Utilizada padrões arquiteturais de Discovery Pattern com Eureka e Centralized Configuration com Spring Cloud Config. Além disso foi utilizado Keycloak para segurança e Restful com padrão MVC para construção das API's.
Obs: algumas partes ainda estão em desenvolvimento.

Para visualizar o código e saber como executar o aplicativo, por favor entre em: Github/Ecommerce
